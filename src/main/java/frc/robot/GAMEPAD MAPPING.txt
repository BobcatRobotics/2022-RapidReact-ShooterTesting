GAMEPAD MAPPING

A:              Climber toggle
B:              Intake + vertical wheels out
X:              Intake solenoid on / Climber solenoid off
Y:              Intake solenoid off / Climber solenoid on
D-Pad right:    Shooter solenoid on / Climber winch unwind
D-Pad left:     Shooter solenoid off
D-Pad down:     Intake wheels out / Climber winch rewind
Left bumper:    Run shooter
Left trigger:   Run tower in
Right bumper:   Intake bar in
Right trigger:  Vertical wheels in
